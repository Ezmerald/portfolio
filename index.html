MaktabaTech Augmented Reality Application

Built an Augmented-Reality based Android application using Unity and C for the stratup MaktabaTech. Application overlays animation, text and audio in the mother language of the child and the language he wants to learn when the camera of the smart device is overlaid with the illustrations of the book. Aim of application is to build children's linguistic, spelling, and translation skills. Application was tested with 100 students.
